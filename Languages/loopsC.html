Loops

if-else statements

The if conditional statement helps us to execute certain commands subject to the condition that a given statement is TRUE.
The general syntax of 'if' statement is given by-
if(condition) {statements;}

eg:
if(a>c)
{
printf("a is greater than c");
}

Here if is a reserved key word. the condition typed inside braces refers to a logical statement. The statement refers to a single or a set of statements seperated by semicolon which will be executed if the condition is true. First the condition is logically evaluated and if it evaluates to TRUE, the statement is executed. If the condition evaluates to FALSE, the statement is not executed.

When the condition in the if statement fails, we can provide an alternate path with an else statement followed by if. The general format is as follows:
if(condition){ statements;} else{statements;}
When condition evaluates to TRUE, statement1 is executed. When it fails, statement2 is executed.


for loop

The for loop is useful for iteratively executing a group of instructions. The general format is given by
for(initialisation;condition;inc/dec){statements;}

eg:
for(i=0;i<=n;i++)
{
print(i)
}

The value of the variable i is iniialised in the first step.The condition is checked, if the condition is true ,then the statements inside te bloc are executed. The expression can be either a simple statement or a set of complex statements which may or may not involve the iterated elements.After the statements are executed once , the iterating variable is either incremented or dercremented and the statements are executed till the condition fails.


while loop

The while loop is used for executing a statement until a condition is valid. The loop terminates when the condition fails. The general format is
while( condition ){ expressions;}

eg:
while(num>0)
{
num=num-10;
}

First the condition is tested. If it is TRUE, the expression is executed, which generally modifies the condition. Then, the condition is again tested. If it is TRUE, the expression is executed. This goes on until the condition fails. When this happens, the while loop is terminated.
