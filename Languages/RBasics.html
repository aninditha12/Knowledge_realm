The Basics

R is a programming language as well as a software environment for statistical computing with excellent libraries for statistical analysis, graphics and data mining. It is maintained under GNU project and is freely available for download. R is designed to be highly extendable through libraries created and added by the users. To execute a file in R, the command 'source(file_name.r)' is used in the terminal.

Input and output

R has many built-in functions and library packages that are readily available. Some of the functions like scan() and print() are widely used for standard input and output operations respectively.

The print() function is used to output data to the standard output device (screen). The syntax of the print() function is:

print(objects_to_print)

The cat() function is similar to the print() function. It is used when we don't want the output number to be printed alongside the output. cat() also recognises escape sequences whereas print() does not. The syntax of the cat() function is:

cat(objects_to_print , file = "", sep = " ", fill = FALSE, labels = NULL,append = FALSE)

The scan() function is used to take in values for variables. R does not have any format specifiers for different data types. The syntax of the scan() function is:

variable_name <- scan()

The readline() function is also used sometimes to input strings. The syntax of the readline() function is:

variable_name <- readline()



Data types

Numeric,integers,complex, logical and character are the basic data types in R.
Decimal values are referred as numeric data types in R. This is the default working out data type.
If you want to create any integer variable in R, you have to invoke the as.integer() function to define any integer type data.
A complex value for coding in R can be defined using the pure imaginary values 'i'.
A logical value is mostly created when comparison between variables are done. eg: 'c=a>b'
A character object can be used for representing string values in R. You have to convert objects into character values using the as.character() function within your code.
Vector, factor, array, matrix, dataframe and list are the further data types. They are widely used to perform various operations.
